# Project Atlas

Official landing page for AaryaRushi Automation Labs.

## Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Local Setup

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually:

```text
http://127.0.0.1:5173
```

## Production Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this `Project_Atlas` folder to a GitHub repository.
2. Open Vercel and choose **Add New Project**.
3. Import the repository.
4. Set the framework preset to **Vite**.
5. Use:
   - Build command: `npm run build`
   - Output directory: `dist`
6. Deploy on the free plan.

After deployment, update the website placeholder and Open Graph URL in `index.html`.
