# AaryaRushi Automation Labs Business Presentation

Professional 16:9 business presentation kit for coaching institutes, schools, training institutes and educational organizations.

## Main File

- `AaryaRushi_Automation_Labs_Business_Presentation.pptx`

## Included Deliverables

- PowerPoint-compatible presentation source
- Editable SVG icon assets
- PNG companion assets for presentation use
- Slide-by-slide animation guide
- Voice-over narration script
- Presenter notes
- Folder structure documentation
- Repeatable Python generator in `source/create_presentation.py`

## Recommended Timing

Target duration: 90-120 seconds.

Suggested pacing:

- Slide 1: 12-15 seconds
- Slide 2: 14-16 seconds
- Slide 3: 18-22 seconds
- Slide 4: 10-12 seconds
- Slide 5: 14-16 seconds
- Slide 6: 10-12 seconds
- Slide 7: 8-10 seconds
- Slide 8: 12-15 seconds

## Animation Style

Use only:

- Fade
- Slide
- Zoom
- Grow

Avoid spinning, bouncing and flashy transitions. Keep all motion subtle and corporate.

## Brand Direction

- Colors: blue, white, dark navy, light gray
- Typography: Inter and Poppins
- Style: clean corporate SaaS presentation
- Icons: minimal editable line icons
